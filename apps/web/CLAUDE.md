# CLAUDE.md — apps/web

Vite + React 19 + TanStack Router (file-based) + TanStack Query + Tailwind v4 + shadcn/ui (Base UI underneath).

## Conventions

- Route with detail page that should NOT inherit parent layout: trailing underscore (`browse_.$category.$slug.tsx`). Breaks layout nesting, keeps URL clean.
- All internal links go through [src/components/link.tsx](src/components/link.tsx) — wraps TanStack `Link`, falls back to `<a>` for external URLs.
- Shared warframe types / client helpers live in [src/lib/warframe.ts](src/lib/warframe.ts). Heavy data is static JSON under `public/data/`, generated by [scripts/build-items-index.ts](../../scripts/build-items-index.ts) (`bun run build:items`).
