## Conventions specific to this app

- Routes with their own detail pages that should NOT inherit the parent layout: use a trailing underscore (`browse_.$category.$slug.tsx`). The underscore breaks layout nesting while keeping the URL path clean.
- Shared warframe types and client helpers live in `src/lib/warframe.ts`. Heavy data (full item records) is static JSON under `public/data/`, generated by `legacy/scripts/build-items-index.ts`.
- All internal links go through `src/components/link.tsx`, which wraps `@tanstack/react-router`'s `Link` and falls back to `<a>` for external URLs.
